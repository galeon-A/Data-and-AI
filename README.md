# Industrial IoT Engine Telemetry Data Quality Platform
Overview

An enterprise-scale Industrial IoT telemetry and data quality platform built using PySpark, Delta Lake, and Databricks.

This project simulates real-world engine telemetry pipelines commonly used in:

Manufacturing
Heavy equipment monitoring
Fleet management
Industrial IoT analytics
Predictive maintenance systems

# About This Project

The Industrial IoT Engine Telemetry Data Quality Platform is designed to simulate enterprise-scale telemetry processing systems used in manufacturing, heavy equipment monitoring, and industrial analytics environments.

The platform ingests high-volume telemetry data from multiple raw formats, processes the data through Bronze and Silver lakehouse layers, and generates Gold-level data quality metrics for operational monitoring.

The project focuses heavily on:

* Data quality engineering
* Sensor validation frameworks
* Industrial telemetry analytics
* Distributed PySpark processing
* Enterprise lakehouse architecture

The system intentionally injects:

* Missing values
* Outliers
* Duplicates
* Multi-format inconsistencies

to simulate real-world industrial telemetry challenges.

The project demonstrates how organizations can build scalable telemetry governance systems capable of validating and monitoring operational sensor data at enterprise scale.

This serves as a portfolio-grade demonstration of:

* Industrial data engineering
* Enterprise data quality frameworks
* Medallion architecture implementation
* IoT telemetry processing pipelines
* Delta Lake engineering workflows

# Pharma Commercial Patient Journey Lakehouse
Overview

An enterprise-scale healthcare and pharmaceutical data engineering platform built using PySpark, Delta Lake, and Databricks architecture principles.

This project simulates a real-world pharmaceutical patient journey ecosystem involving:

Patient HUB programs
Claims processing
Specialty pharmacy shipments
Provider master data
Commercial analytics pipelines

# About This Project

The Pharma Commercial Patient Journey Lakehouse simulates a real-world pharmaceutical commercial analytics ecosystem using modern lakehouse architecture principles.

The platform processes patient, claims, provider, and specialty pharmacy datasets through Bronze and Silver layers using PySpark and Delta Lake.

The goal of the project is to showcase:

* Healthcare data engineering workflows
* Enterprise ETL design
* Data quality handling
* PHI governance concepts
* Commercial analytics pipeline development

This project reflects how modern pharmaceutical organizations manage:

* Patient support programs
* Claims adjudication
* Provider networks
* Specialty pharmacy fulfillment
* Commercial operations analytics

The architecture is intentionally designed to mirror enterprise healthcare systems used in:

* Pharma manufacturers
* Specialty pharmacies
* Commercial analytics platforms
* Patient HUB operations
* Healthcare data modernization initiatives

The project also demonstrates scalable engineering patterns including:

* Schema enforcement
* Deduplication logic
* Data standardization
* Delta Lake processing
* Governance-aware transformations
* Audit tracking
* Lineage enrichment
